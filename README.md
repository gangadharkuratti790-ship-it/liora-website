# Liora Cosmetic Products Website

## 📌 Project Overview

Liora is a cosmetic products website created using coding languages.
This project was developed as part of a workshop to learn modern web design and front-end development.

---

## Features

* Attractive clicks of cosmetic products
* Product display sections
* Simple and clean structure

---

## Technologies Used

* HTML5
* CSS3
